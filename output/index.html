I Can Hold My Breath Forever

Controls:

  • Arrows to move
  • P to pause
  • + or - to change volume
  • 0 to mute

This game was designed+developed by Jake Elliott as an entry in the '10 Seconds' friendly game competition organized by the Experimental Gameplay Project. It was built using the flixel engine by Adam Saltsman.

The full source code for this game is available on github (Copy-It-Right).

The compiled version uses the proprietary pixel font 'Gardenia' by Kemie Guaida (licensed).
